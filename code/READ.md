# Grafter

Grafter is a GUI tool to examine behavior differences between clones.

## Compilation

Grafter can be compiled with the command:

    $ ant

or with maven

    $ mvn clean install

## Usage

Start grafter with the command:

    $ java -jar Grafter.jar
